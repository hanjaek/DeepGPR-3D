# (권장) 가상환경 + 의존성 설치를 먼저 했다면---------------------------------

python detect.py \
  --weights "../2) 학습모델 파일/YZ/best.pt" \
  --source "../내_이미지_폴더/" \
  --imgsz 640 \
  --conf-thres 0.4 \
  --device 0 \
  --save-txt --save-conf

폴더 안에 GPR 단면 이미지 파일(.jpg, .png, .jpeg, .bmp 등) 만 넣어두면 됨.
파일 이름은 아무 이름이나 상관 X

단,
YZ 모델은 YZ 단면만 인식
XY 모델은 XY 단면만 인식
XZ 모델은 XZ 단면만 인식

가능한 수준: “지하 구조의 형태를 3D로 시각화”
지금 가진 출력물(JSON 또는 .txt 바운딩박스) 만으로 충분
